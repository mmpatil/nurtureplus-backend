## FAQ

**Q: Should I change the response to return Firebase UID as user_id?**  
A: No! Your current design is correct. Internal UUID isolates your system from Firebase changes.

**Q: Is the token safe in logs?**  
A: Yes! Only firebase_uid is logged, never the token itself.

**Q: What if the token expires?**  
A: iOS will get 401, refresh the token with `getIDToken(forceRefresh: true)`, and retry.

**Q: Can hackers create fake tokens?**  
A: No! Firebase Admin SDK verifies the signature using Firebase's public keys.

**Q: Do I need to refresh tokens?**  
A: Yes, every hour. iOS handles this automatically with `getIDToken()`.
